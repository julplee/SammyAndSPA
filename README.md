SammyAndSPA
===========

Adding Sammy.js on the Microsoft SPA template. This project is our first test implementing Sammy.js in a .Net SPA project.


== Requirements ==
* Visual Studio 2012
* ASP.Net MVC4
* ASP.Net WebApi


== Build for .NET 4.0 ==

NinjectWithMvcAndWebApi is built for .NET 4.0 primarily.
